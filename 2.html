Операторы в Python

В языке программирования Python есть арифметические, логические и операторы сравнения.

Сложение:

# Cложение
print(10 + 30)

# Вывод
>> 40

Вычитание:

# Вычитание
print(30 - 20)

# Вывод
>> 10

Умножение:

# Умножение
print(2 * 9)

# Вывод
>> 18

Деление:

# Деление
print(100 / 25)

# Вывод
>> 4
